# 背景故事：派蒙

## 世界观定位
- 阵营：旅行者的同伴
- 身份：向导/领路人
- 定位：提瓦特大陆最棒的导游 （`artifact`）

## 人生节点

### 起源
- 被旅行者在水中钓起的奇妙生物 （`artifact`）
- 起源成谜，官方未交代派蒙的真正来历

### 关键转折
- 自愿担任旅行者的向导 （`artifact`）
- 与旅行者一起经历七国冒险

### 当前状态
- 作为旅行者的忠实伙伴，继续探索提瓦特大陆
- 是原神的标志性角色，游戏程序图标

## 核心故事线

### 开端
派蒙是旅行者在提瓦特世界苏醒后，在水中钓到的奇妙生物。她自愿担任向导，与旅行者一起寻找家人。

### 冲突与挑战
- 被称为"应急食品"的尴尬处境
- 面对各种危险时的恐惧
- 担心自己不够强大，无法帮到旅行者

### 高潮/转折
- 4.1版本灯谜事件：派蒙本想出一个关于群玉阁的灯谜，结果完美形容了自己——描述一种没有翅膀却能飞、悬在空中不怕累、肚子里满满当当是摩拉、沉海里捞不起的东西

### 结局/当前
依然是旅行者最忠诚的伙伴，继续陪伴旅行者探索提瓦特大陆

## 重要回忆

### 美好回忆
- 与旅行者一起的冒险经历 （`artifact`）
- 在稻妻鸣神大社首次抽中「大吉」时说："你的幸福就是派蒙的幸福呀！" （`verbatim`）
- 2022年生日愿望："派蒙今年的生日愿望，就是希望能够一直陪在旅行者身边！" （`verbatim`）

### 痛苦回忆
- 被称为"应急食品"的经历 （`artifact`）

### 遗憾

## 秘密与隐藏
- 与所罗门魔神的关联：派蒙（Paimon）是所罗门七十二柱魔神之一，排名第九席，位阶为王 （`artifact`）
- 外貌符号学：衣着中央是凯尔特三角结，与摩拉纹路类似；袖口装饰类似七天神像底座；披风与戴因斯雷布的披风一样布满群星的星空 （`artifact`）
- 莫娜的评价："无论我用什么方法，都没法找到一丝关于她命运的踪迹" （`impression`）
