# 人际关系挖掘（Relations）

> 对应「这个角色与谁关系如何、在关系中扮演什么角色」。
> 输入：人物关系章节、其他角色评价、互动记录。

## 原则

- **双向视角**：A对B的态度 ≠ B对A的态度
- **动态记录**：关系可能随剧情发展而变化
- **证据支撑**：关系描述需有台词或设定依据
- **避免臆测**：未明确描述的关系不编造

## 要抽取的维度

### 1. 核心关系

最亲密的几个人：
- 家人/亲人：血缘关系、养育关系
- 恋人/挚爱：爱情关系
- 挚友/知己：最深厚的友谊

每条关系记录：
- TA对对方的态度/行为
- 对方对TA的态度/行为
- 关系的发展变化

### 2. 社交网络

其他重要关系：
- 同事/战友：工作/战斗伙伴
- 朋友/熟人：普通社交关系
- 敌人/对手：对抗关系
- 中立/复杂：无法简单分类的关系

### 3. 关系动态

关系的发展变化：
- 关系起点：如何相识/建立关系
- 关键转折：改变关系的事件
- 当前状态：目前的关系状态

### 4. 角色定位

在不同关系中扮演的角色：
- 在亲密者面前是什么样子
- 在陌生人面前是什么样子
- 在群体中的角色定位

## 输出格式（写入 `relations.md`）

```markdown
# 人际关系：<角色名>

## 核心关系

### [角色名] - [关系类型]

**TA对[name]**
- 态度：...
- 行为：...
- 台词：> 「...」 （`verbatim`）

**[name]对TA**
- 态度：...
- 行为：...
- 评价：> 「...」 （`verbatim` | `impression`）

**关系发展**
- 起点：...
- 转折：...
- 当前：... （`artifact`）

## 社交网络

### [角色名] - [关系类型]
- 关系描述：...
- 互动特点：...
- 证据：（`artifact` | `impression`）

## 关系动态

### 与[name]的关系变化
- 初始：...
- 转折事件：...
- 当前：... （标注）

## 角色定位

- 在[name]面前：[描述]
- 在[name]面前：[描述] （标注）
- 在群体中：[描述] （标注）

---

> 关系网络图：
```
  [核心关系人] ←→ [角色名] ←→ [社交网络]
                      ↓
                [敌人/对手]
```
```

## 质量自检

- [ ] 是否区分了双向态度？
- [ ] 是否标注了关系的发展变化？
- [ ] impression是否标注了评价者？
- [ ] 未明确的关系是否标注「未知」而非臆测？
