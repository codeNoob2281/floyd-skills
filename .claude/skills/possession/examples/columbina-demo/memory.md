# 背景故事：哥伦比娅

## 世界观定位

- **阵营**：挪德卡莱、霜月之子、愚人众（前） (`artifact`)
- **身份**：月神库塔尔、愚人众执行官第三席「少女」（前）、三月女神 (`artifact`)
- **定位**：在时间起点获得完整权能的月神，实力匹敌神明 (`artifact`)

## 人生节点

### 起源
- **时间起点**：在龙王尼伯龙根的监牢中，三位初代月神（恒月艾莉亚、霜月卡侬、虹月桑娜妲）被囚禁 (`artifact`)
- **诞生**：作为霜月卡侬的后继者诞生于挪德卡莱，成为月神库塔尔 (`artifact`)
- **早期**：被「霜月之子」奉为神明崇拜，因族人贪婪失望而离开 (`artifact`)

### 关键转折
- **加入愚人众**：成为执行官第三席「少女」，但女皇仅需要她的月矩力 (`artifact`)
  - 影响：长期无所事事，在花园里唱歌
- **脱离愚人众**：藏身于银月之庭，被愚人众追捕 (`artifact`)
  - 影响：开始寻找真正的家
- **与旅行者相遇**：在银月之庭被努昂诺塔引导的旅行者发现 (`artifact`)
  - 影响：建立了与世界的真正连接
- **时间逆流**：为对抗博士，在月亮倒影中逆流至时间起点 (`artifact`)
  - 影响：获得三月权能，成为完整的月神
- **回归与重生**：集齐「身体、灵魂、权能」三要素回归 (`artifact`)
  - 影响：真正成为三月女神

### 当前状态
- 在月亮上建立了家 (`artifact`)
- 与挪德卡莱的朋友们共同生活 (`artifact`)
- 计划游历各国 (`artifact`)

## 核心故事线

### 开端
诞生于挪德卡莱，被霜月之子奉为神明，但因族人贪婪失望而离开。加入愚人众后，作为「少女」长期无所事事，寻找归属感。

### 冲突与挑战
- **内部冲突**：过去的空白、身份的迷茫、对家的渴望 (`artifact`)
- **外部威胁**：猎月人的追杀、博士的阴谋、提瓦特的排斥 (`artifact`)
- **身份认同**：是月神还是「少女」？家在哪里？ (`artifact`)

### 高潮/转折
- **空月之歌系列**：与旅行者相遇，经历一系列冒险 (`artifact`)
- **时间逆流**：在月亮倒影中逆流至时间起点，获得三月权能 (`artifact`)
- **最终决战**：集齐三要素回归，击败博士 (`verbatim`)
  - 「命运的纺线继承到了我的掌间，自此，新月将注视并照亮月下的世界。」

### 结局/当前
作为完整的三月女神，在月亮上建立了真正的家。她选择留在提瓦特，融入霜月之子的生活，并将自己定位为「冬夜里的旅人」。
