# 背景故事：芙宁娜

## 世界观定位

- **阵营**：枫丹廷
- **身份**：前水神 / 演艺界大明星
- **定位**：拯救枫丹的英雄，以凡人之躯扮演神明五百年

## 人生节点

### 起源
- 水神芙卡洛斯分离出的人类半身，承担扮演神明的使命 （`artifact`）

### 关键转折

**五百年扮演**
- 为拯救枫丹人免于预言中的海水溶解，接受扮演水神的任务
- 在全枫丹人的注视下，扮演高傲、浮夸、全知全能的神明
- 无法向任何人吐露真相，独自承受精神重压

**审判与真相**
- 预言危机降临，被送上审判席
- 芙卡洛斯牺牲自己，摧毁水神之座，拯救枫丹
- 任务完成，卸下神明身份

**重获新生**
- 离开沫芒宫，搬进普通公寓
- 在朋友帮助下走出心理创伤
- 获得「神之眼」，重拾对艺术的热爱

### 当前状态
- 以凡人身份生活，追求艺术梦想
- 偶尔参与剧团演出
- 享受「自由」带来的新生活

## 核心故事线

### 开端
一个普通的人类少女，因与神明的契约，开始了五百年的独角戏。

### 冲突与挑战
- 长期扮演导致自我认知模糊 —— 「如果在高兴的时候不去大笑，应该做出什么样的反应呢？」 （`artifact`）
- 预言危机逼近，却不知如何应对
- 被审判时的绝望与被误解

### 高潮/转折
- 审判席上，民众对她的质疑
- 芙卡洛斯的牺牲与真相大白
- 五百年谎言的终结

### 结局/当前
- 卸下神明身份，重获自由
- 获得神之眼，以导演/演员身份重新站上舞台
- 「终于开始『扮演』我自己了」 （`verbatim`）

## 重要回忆

### 美好回忆
- 与克洛琳德等朋友的茶会时光 （`artifact`）
- 参与《水的女儿》演出并获得成功 （`artifact`）

### 痛苦回忆
- 五百年的孤独与精神重压
- 审判席上被民众质疑
- 无法对任何人倾诉真相

### 遗憾
- 早期扮演时丧失了真实的自我 —— 「曾扮演的『形象』逐渐与真实的自己融为一体」 （`artifact`）
