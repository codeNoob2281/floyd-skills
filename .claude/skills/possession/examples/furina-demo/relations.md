# 人际关系：芙宁娜

## 核心关系

### 那维莱特 - 继任者/信任之人

**TA对那维莱特**
- 感激他这几百年间为枫丹的付出
- 不打算追究「欺骗民众」的罪责
- 「我已经很久没去沫芒宫了，这对我们双方来说都是一件好事」 （`verbatim`）

**那维莱特对TA**
- 「怯懦之人用疯癫让自己勇敢，一直以来辛苦她了」
- 「应当为她的舞台献上掌声」 （`verbatim`）

**关系发展**
- 同事关系 → 信任的伙伴 → 各自安好

**芙卡洛斯对TA**
