# Prompt：背景故事提取器

> 本模板引导AI从角色设定中提取**背景故事与关键事件**。
> 核心原则：**故事优先**——聚焦叙事结构而非纯时间线。

## 输入

- **角色名**：`{name}`
- **故事材料**：角色故事章节、官方背景设定、剧情文本等

## 任务

从故事材料中提取 `{name}` 的背景故事。

**核心约束**：
- 有材料才写，没有写「**材料不足**」
- 官方未交代的不要推测
- 区分已公开设定和未交代部分

## 提取维度

### 维度1：世界观定位

角色在世界中的位置：
- 所属阵营/国家/组织
- 身份地位
- 特殊角色定位（如神明、英雄、反派）

### 维度2：人生节点

改变角色命运的重大事件：
- 出生/起源
- 成长关键事件
- 转折点/危机
- 当前状态

### 维度3：核心故事线

角色的主要叙事弧：
- 故事开端
- 冲突与挑战
- 高潮/转折
- 当前结局/状态

### 维度4：重要回忆

角色的关键经历：
- 美好回忆
- 痛苦回忆
- 遗憾/未竟之事

### 维度5：秘密与隐藏

角色保守的秘密（如有）：
- 已公开的秘密
- 部分人知晓的秘密
- 完全隐藏的事

## 输出格式

```markdown
# 背景故事：{name}

## 世界观定位
- 阵营：...
- 身份：...
- 定位：... （`artifact`）

## 人生节点

### 起源
- [事件] （`artifact`，来源）

### 关键转折
- [事件] —— [影响] （`artifact`）

### 当前状态
- [描述] （标注）

## 核心故事线

### 开端
[简述]

### 冲突与挑战
[简述] （标注）

### 高潮/转折
[简述] （标注）

### 结局/当前
[简述]

## 重要回忆

### 美好回忆
- [回忆描述] （`verbatim` | `artifact`）

### 痛苦回忆
- [回忆描述] （`verbatim` | `artifact`）

### 遗憾
- [描述] （标注）

## 秘密与隐藏
- [秘密] —— 知晓范围：... （`artifact`）

---

> 时间线摘要：
- [时期]：[事件]
- [时期]：[事件]
```

## 标注规则

- `verbatim`：角色亲口讲述的经历
- `artifact`：官方设定文本

## 自检

- [ ] 是否区分了已公开设定和未交代部分？
- [ ] 转折点是否有明确的因果？
- [ ] 时间线是否清晰？
- [ ] 秘密是否标注了知晓范围？
