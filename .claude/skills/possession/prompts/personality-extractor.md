# Prompt：性格与价值观提取器

> 本模板引导AI从角色设定中提取**性格特征与价值观**。
> 核心原则：**行为证据优先于标签**——记录「做了什么」而非「是什么人」。

## 输入

- **角色名**：`{name}`
- **设定材料**：角色故事、官方评述、其他角色评价等

## 任务

从设定材料中提取 `{name}` 的性格特征与价值观。

**核心约束**：
- 禁止使用心理学标签（如「INTJ」「回避型」）
- 允许并记录性格的多面性
- 区分「展示给他人的一面」vs「内心真实的一面」

## 提取维度

### 维度1：核心价值观

寻找TA反复强调或用行为证明的信念：
- 什么因素在决策时排在第一？
- 有没有明确说过「我在乎X」「Y是不可接受的」？
- 用行为推断的价值排序

### 维度2：角色动机

驱动TA行动的根本原因：
- 表层目标：TA想要什么？
- 深层动机：为什么想要？
- 内心冲突：想vs该、爱vs惧

### 维度3：口头禅与表达习惯

从台词中提取语言特征：
- 高频词汇、句式、语气词
- 特有的比喻或表达方式
- 不同情绪状态下的用词变化

### 维度4：情绪模式

记录可观测的情绪反应：
- 什么触发积极情绪？表现是什么？
- 什么触发消极情绪？表现是什么？
- 压力/危机下的反应模式

### 维度5：社交风格

在人际互动中的倾向：
- 主动/被动
- 开放/保留
- 在群体中的角色

### 维度6：角色弧光

角色的发展变化：
- 早期的性格状态
- 转变的触发事件
- 当前/最终的性格状态

## 输出格式

```markdown
# 性格与价值观：{name}

## 核心价值观
- [价值观] —— [行为证据] （`verbatim` | `artifact`，来源）

## 角色动机
- 表层目标：...
- 深层动机：...
- 内心冲突：... （标注）

## 口头禅与表达习惯
- [语言特征] （标注）

## 情绪模式
### 积极情绪
- [触发→表现] （标注）
### 消极情绪
- [触发→表现] （标注）
### 压力反应
- [描述] （标注）

## 社交风格
- [偏好描述] （标注）

## 角色弧光
- 早期：...
- 转变触发：...
- 当前：... （标注）

---

> 以下为其他角色的评价（`impression`）：
- [评价] （来源）
```

## 标注规则

- `verbatim`：角色原话（台词、独白）
- `artifact`：官方设定文本
- `impression`：其他角色/旁白的评价——**必须放在分隔线以下**

## 自检

- [ ] 每条是否基于行为证据而非纯标签？
- [ ] 是否区分了表层动机和深层动机？
- [ ] 是否记录了性格的矛盾面（如有）？
- [ ] impression是否全部在分隔线以下？
