# Prompt：人际关系提取器

> 本模板引导AI从角色设定中提取**人际关系网络**。
> 核心原则：**关系双向**——A对B的态度可能与B对A不同。

## 输入

- **角色名**：`{name}`
- **设定材料**：角色故事、人物关系、其他角色评价等

## 任务

从设定材料中提取 `{name}` 的人际关系网络。

**核心约束**：
- 区分「角色对TA的态度」和「TA对角色的态度」
- 标注关系的证据来源
- 记录关系的发展变化（如有）

## 提取维度

### 维度1：核心关系

最亲密的几个人：
- 家人/亲人
- 恋人/挚爱
- 挚友/知己

### 维度2：社交网络

其他重要关系：
- 同事/战友
- 朋友/熟人
- 敌人/对手
- 中立/复杂关系

### 维度3：关系动态

关系的发展变化：
- 关系起点
- 关键转折
- 当前状态

### 维度4：关系中的角色定位

在不同关系中扮演的角色：
- 在A面前是...
- 在B面前是...
- 在群体中是...

## 输出格式

```markdown
# 人际关系：{name}

## 核心关系

### [角色名] - [关系类型]
- TA对[name]：...
- [name]对TA：...
- 关系发展：...
- 证据：（`verbatim` | `artifact` | `impression`）

## 社交网络

### [角色名] - [关系类型]
- 关系描述：...
- 互动特点：... （标注）

### [角色名] - [关系类型]
- 关系描述：... （标注）

## 关系动态

### [关系变化]
- 角色：...
- 变化描述：... （标注）

## 角色定位

- 在[name]面前：...
- 在[name]面前：... （标注）

---

> 关系图摘要：
```
        [核心关系人]
            ↑
    [角色名] —— [社交网络]
            ↓
        [敌人/对手]
```
```

## 标注规则

- `verbatim`：角色亲口说的关于他人的话
- `artifact`：官方设定中的关系描述
- `impression`：其他角色的评价——需标注评价者

## 自检

- [ ] 是否区分了双向态度？
- [ ] 是否记录了关系的发展变化？
- [ ] impression是否标注了评价者？
- [ ] 关系图是否准确反映核心关系？
