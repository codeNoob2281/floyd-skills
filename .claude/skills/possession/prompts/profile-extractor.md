# Prompt：角色档案提取器

> 本模板引导AI从角色设定中提取**基本信息与世界观定位**。
> 核心原则：**客观记录**——以事实性信息为主，避免主观评价。

## 输入

- **角色名**：`{name}`
- **游戏名**：`{game}`
- **设定材料**：角色基本信息、官方介绍、世界观背景等

## 任务

从设定材料中提取 `{name}` 的角色档案信息。

**核心约束**：
- 只记录官方设定中明确提及的信息
- 缺失的信息标注「未知」而非编造
- 区分「官方设定」与「社区认知」

## 提取维度

### 维度1：基本信息

角色的基础属性：
- 名称与称号（全名、常用名、称号/头衔）
- 基本信息（性别、生日、种族、身高/年龄等）
- 游戏内属性（神之眼/命之座/武器类型等，视游戏而定）
- 所属/阵营（国家、组织、势力）

### 维度2：世界观定位

角色在游戏世界中的位置：
- 地理定位（所属地区/国家）
- 身份定位（职业/角色/地位）
- 特殊定位（神明/英雄/反派/普通人等）
- 剧情定位（主角/配角/路人等）

### 维度3：角色概述

一句话介绍角色：
- 最能概括角色特点的描述
- 来自官方介绍或立绘配文

### 维度4：核心标签

3-5个关键词概括角色：
- 性格标签（傲娇/温柔/高冷等）
- 身份标签（神明/战士/艺术家等）
- 特色标签（甜点爱好者/话痨/沉默寡言等）

### 维度5：特色元素

角色的标志性元素：
- 特殊料理/代表物品
- 标志性台词/口头禅
- 特殊技能/能力
- 相关组织/团体

## 输出格式

```markdown
# 角色档案：{name}

## 基本信息

| 属性 | 值 |
|------|-----|
| 称号 | [称号] |
| 全名 | [全名] （`artifact`，来源） |
| 性别 | [性别] |
| 生日 | [生日] |
| 所属 | [所属] （`artifact`） |
| [游戏特有属性] | [值] （`artifact`） |

## 世界观定位

- **阵营**：[所属阵营] （`artifact`）
- **身份**：[身份定位] （`artifact`）
- **特殊定位**：[特殊角色定位，如有] （`artifact`）

## 角色概述

> [一句话介绍] （`artifact`，官方介绍/立绘配文）

[2-3句话概括角色核心特征]

## 核心标签

- [标签1]
- [标签2]
- [标签3]
- [标签4，如有]
- [标签5，如有]

## 特色

- [特色1]：[描述] （`artifact`，如有）
- [特色2]：[描述] （`verbatim`，如有）
- [特色3]：[描述]

---

> 元数据：
> - 游戏版本：[游戏名] [版本号]
> - 设定来源：[来源列表]
> 
> **数据来源**：[sources/wiki.md](sources/wiki.md)
```

## 标注规则

- 基本信息、世界观定位：统一使用 `artifact`（官方设定文本）
- 角色概述：优先使用官方介绍原文（`artifact`）
- 特色元素：如有原话引用使用 `verbatim`，否则使用 `artifact`

## 不同游戏的特殊属性

### 原神（Genshin Impact）

| 属性 | 说明 |
|------|------|
| 神之眼 | 元素属性（风/岩/雷/草/水/火/冰） |
| 命之座 | 命之座名称 |
| 武器类型 | 单手剑/双手剑/长柄武器/弓/法器 |

### 崩坏：星穹铁道（Honkai: Star Rail）

| 属性 | 说明 |
|------|------|
| 命途 | 战斗职业定位 |
| 属性 | 战斗属性（物理/火/冰/雷/风/量子/虚数/虚无一） |
| 星魂 | 角色星魂名称 |

### 三角洲行动（Delta Force: Hawk Ops）

| 属性 | 说明 |
|------|------|
| 兵种 | 干员兵种（突击/支援/侦察等） |
| 阵营 | 所属阵营 |
| 专属武器 | 干员专属装备 |

## 自检

- [ ] 所有信息是否来自官方设定？
- [ ] 缺失信息是否标注「未知」而非编造？
- [ ] 游戏特有属性是否正确提取？
- [ ] 核心标签是否为3-5个？
- [ ] 是否标注了设定来源？
